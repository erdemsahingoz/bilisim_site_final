# Bilişim Sitesi — Tech Store

Bilişim ürünleri e-ticaret demo sitesi.

## Teknolojiler
- React (functional components)
- React Router DOM
- Material UI (MUI)

## Ürünler
- Kingston NVMe SSD
- Seagate BarraCuda HDD 2TB
- Corsair Vengeance DDR4 RAM
- Logitech G203 Oyuncu Mouse
- Mekanik RGB Klavye
- Gaming PC Kasa (Mid-Tower)
- Gaming Laptop 15"
- Samsung Curved Gaming Monitör 27"

## Çalıştırmak için

```bash
cd frontend
npm install
npm start
```

Tarayıcıda aç: http://localhost:3000

## Sayfalar
- `/` → Ana sayfa (ürün listesi)
- `/products/:id` → Ürün detay sayfası
